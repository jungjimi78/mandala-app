# 성장 만트라 앱

GitHub Pages에 올리는 방법:

1. 이 ZIP 파일을 풀기
2. 안에 있는 파일 전부를 GitHub repository의 root에 업로드
   - index.html
   - style.css
   - app.js
   - manifest.json
   - icon.png
   - icon-192.png
   - icon-512.png
3. Settings → Pages
4. Source: Deploy from a branch
5. Branch: main / root
6. Save
7. 1~2분 후 `https://사용자이름.github.io/레포이름/` 접속

아이폰 앱처럼 쓰기:
Safari에서 링크 열기 → 공유 버튼 → 홈 화면에 추가
